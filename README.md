# USGS Earthquake Activity 

## Overview

The Jupyter notebook retrieves GeoJSON past 24-hour summary data from the [U.S. Geological Survey](https://earthquake.usgs.gov/earthquakes/feed/v1.0/geojson.php) for data analysis.
